# macOS configuration

Auto configuration: `sh setup_mac.sh`

## Finder

### Shortcuts

- Copy directory/file path `CMD + Option + C`
- Quick search  `CMD + Shift + G`

### Options (`CMD + J`)

- Text size `12`
- Icon size `54x54`
- Grid spacing `Fourth line`
- Show item info
- Show icon preview

### View

- Show Path Bar
- Show Status Bar
